##  Getting Started

To run any of the projects locally:

```bash
git clone https://github.com/JoshiDeepak08/<REPO_NAME>.git
cd <REPO_NAME>
````

Install required dependencies (example):

```bash
pip install -r requirements.txt
```

Then open relevant Jupyter notebooks:

```bash
jupyter notebook
```
